A Fabric mod for Minecraft that add an aspen biome to game.
You can download a ready-made [JAR](https://github.com/dungeonerr/Aspengrove-Mod/releases) file, or build it yourself by doing this:

Download the ZIP by clicking the green button `Code` -> `Download ZIP`

Next, unzip the ZIP file.

And follow the steps described in the [Fabric Documentation](https://docs.fabricmc.net/develop/getting-started/building-a-mod).
